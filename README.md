# HYROX Planner App

A Vite + React app based on the HYROX training planner logic developed from the spreadsheet workflow.

## Features

- benchmark inputs for row, ski, run, FTP, and strength
- date-driven programme length
- automatic phase allocation
- benchmark-driven pacing
- strength progression by phase
- optional Saturday HYROX class logic
- dashboard and weekly plan view

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
